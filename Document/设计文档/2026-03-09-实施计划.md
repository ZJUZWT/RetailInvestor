# 散户大冒险 - 第一版实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 构建一个可玩的炒股主题Roguelike网页游戏MVP，包含完整回合流程、股价模拟、事件系统、卡牌系统和交易UI。

**Architecture:** 纯前端SPA，React 18 + TypeScript + Vite。游戏引擎层（engine/）负责回合流程、股价模拟、事件和卡牌逻辑；状态层（stores/）用Zustand管理全局游戏状态；UI层（components/）负责渲染K线图、交易面板、事件展示等。数据层（data/）用JSON配置事件/卡牌/目标。

**Tech Stack:** React 18, TypeScript, Vite, Zustand, lightweight-charts, Tailwind CSS 4, localStorage

## Task 2: 核心类型定义

**Files:**

**Step 1: 定义所有游戏核心类型**

```typescript
// 游戏阶段
export type GamePhase =
  | 'morning_news'   // 晨报
  | 'am_trading'     // 上午交易
  | 'lunch_break'    // 午间休息
  | 'pm_trading'     // 下午交易
  | 'after_hours'    // 盘后活动
  | 'settlement';    // 今日结算

export type GameStatus = 'menu' | 'playing' | 'won' | 'lost';

// 股票数据点（K线）
export interface StockDataPoint {
  day: number;
  open: number;
  close: number;
  high: number;
  low: number;
}

// 事件稀有度
export type Rarity = 'common' | 'rare' | 'legendary';

// 事件效果
export interface EventEffect {
  type: 'stock_trend' | 'cash' | 'daily_expense' | 'stamina' | 'card' | 'chain';
  value: number;        // 数值（正/负）
  cardId?: string;      // 如果是卡牌效果
  chainEventId?: string; // 如果是事件链
  description?: string;
}

// 游戏事件
export interface GameEvent {
  id: string;
  title: string;
  description: string;
  imageUrl?: string;
  rarity: Rarity;
  effects: EventEffect[];
  chainNext?: string[];
  phase?: GamePhase;     // 在哪个阶段触发
}

// 卡牌效果
export interface CardEffect {
  type: 'info_accuracy' | 'trend_hint' | 'sell_bonus' | 'expense_reduce' | 'stamina_bonus' | 'trade_override';
  value: number;
  description: string;
}

// 卡牌
export interface Card {
  id: string;
  name: string;
  emoji: string;
  description: string;
  rarity: Rarity;
  type: 'passive' | 'active';
  effects: CardEffect[];
  usesRemaining?: number;
}

// 目标
export interface Goal {
  id: string;
  title: string;
  targetAmount: number;
  description: string;
}

// 午间/盘后活动
export interface Activity {
  id: string;
  name: string;
  emoji: string;
  staminaCost: number;
  description: string;
}

// 牛熊周期段
export interface TrendSegment {
  startDay: number;
  endDay: number;
  dailyBias: number;    // 每日基础涨跌偏差 (如 +0.005 = 0.5%/天)
  volatility: number;   // 波动率
}

// 游戏状态
export interface GameState {
  // 基本
  day: number;
  phase: GamePhase;
  gameStatus: GameStatus;

  // 玩家属性
  cash: number;
  shares: number;
  shareCostBasis: number; // 持仓成本
  stamina: number;
  maxStamina: number;
  dailyExpense: number;

  // 目标
  goal: Goal;

  // 股票
  currentPrice: number;
  stockHistory: StockDataPoint[];
  trendSegments: TrendSegment[];
  stockName: string;

  // 交易限制
  boughtToday: boolean;   // T+1: 今日是否已买入
  todayOpen: number;      // 今日开盘价
  amClose: number;        // 上午收盘价（午间参考）

  // 事件 & 卡牌
  cards: Card[];
  maxCardSlots: number;
  activeEventChains: { eventId: string; nextDay: number }[];
  currentEvent: GameEvent | null;
  eventLog: { day: number; event: GameEvent }[];

  // 消息
  messages: string[];
}
```

**Step 1: 实现游戏引擎**
