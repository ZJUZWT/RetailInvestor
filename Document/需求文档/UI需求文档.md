# 散户大冒险 - UI 需求文档

> 最后更新: 2026-03-09

## 1. 设计风格

| 项目 | 描述 |
|------|------|
| 主题 | 深色主题（股市经典黑底） |
| 配色 | 涨红跌绿（A股惯例） |
| 稀有度颜色 | 白色(常见) / 蓝色(稀有) / 金色(传说) |
| 字体 | 系统默认等宽字体（数字） + 系统 UI 字体（文字） |

## 2. 已实现页面

### 2.1 主菜单

- 游戏标题 "散户大冒险"
- "新游戏" 按钮
- "继续游戏" 按钮（有存档时显示）

### 2.2 游戏主界面（桌面端布局）

```
┌────────────────────────────────────────────────────┐
│  散户大冒险  Day 42  ⚡3  💰¥123,456  🎯¥500,000  │  ← 状态栏
├──────────────────────────────┬─────────────────────┤
│                              │ 📰 今日事件          │
│      📈 K线图/分时图区域      │ [事件卡片+配图]      │
│      (lightweight-charts)    │                     │
│                              │ 🃏 我的卡牌          │
│                              │ [卡1] [卡2] [卡3]   │
├──────────────────────────────┤                     │
│  当前价/持仓/可用资金         │ 📋 信息面板          │
│  [买入] [卖出] [观望]        │                     │
│  [全仓] [半仓]              │                     │
└──────────────────────────────┴─────────────────────┘
```

### 2.3 阶段面板

- 晨报面板：昨日复盘 + 隔夜事件 + "开盘" 按钮
- 交易面板：分时图 + 买卖操作 + 速度控制
- 午休面板：信息获取 + "下午开盘" 按钮
- 盘后活动面板：5 种活动选择 + 体力显示
- 结算面板：当日汇总 + 胜负判定

### 2.4 游戏结束

- 胜利/失败提示
- 统计数据（存活天数、峰值资产、最终资产）
- "再来一局" 按钮
