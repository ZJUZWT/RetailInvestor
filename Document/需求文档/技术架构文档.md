# 散户大冒险 - 技术架构文档

> 最后更新: 2026-03-09

## 1. 技术栈

| 类别 | 技术 | 版本 |
|------|------|------|
| 前端框架 | React | 19.2 |
| 语言 | TypeScript | 5.9 |
| 构建工具 | Vite | 7.3 |
| 状态管理 | Zustand | 5.0 |
| K线图 | lightweight-charts | 5.1 |
| 样式 | Tailwind CSS | 4.2 |
| 存档 | localStorage | - |
| 部署 | GitHub Pages (gh-pages) | - |

## 2. 项目结构

```
src/
├── types/index.ts          # 核心类型定义
├── data/                   # 游戏数据（纯数据配置）
│   ├── events.ts           # 32 个事件
│   ├── cards.ts            # 8 张卡牌
│   ├── goals.ts            # 5 个目标
│   └── activities.ts       # 5 种活动
├── engine/                 # 游戏引擎（纯逻辑，不依赖 React）
│   ├── StockSimulator.ts   # 股价生成（趋势段 + 波动率）
│   ├── IntradaySimulator.ts# 日内分时模拟（241 tick/天）
│   ├── EventSystem.ts      # 事件抽取 + 效果应用
│   └── CardSystem.ts       # 卡牌管理
├── stores/                 # 状态管理
│   └── gameStore.ts        # Zustand 全局 store（含实时 tick 播放）
├── components/             # UI 组件（12 个）
├── App.tsx                 # 主入口，阶段路由
└── main.tsx                # 挂载点
```

## 3. 架构分层

```
┌─────────────────────────────────┐
│          UI 层 (components/)     │  React 组件，消费 store 状态
├─────────────────────────────────┤
│        Store 层 (stores/)        │  Zustand，包装引擎为响应式状态 + 定时器
├─────────────────────────────────┤
│       Engine 层 (engine/)        │  纯函数逻辑，不依赖 React
├─────────────────────────────────┤
│        Data 层 (data/)           │  静态游戏数据配置
├─────────────────────────────────┤
│       Types 层 (types/)          │  TypeScript 类型定义
└─────────────────────────────────┘
```

## 4. 关键设计

### 4.1 股价模拟

- 每局预生成 3-5 个趋势段（牛/熊/震荡）
- 日内 241 tick：gravity + noise + trendPush + momentum
- 8 时段不同波动率特征
- 事件修正叠加在基础趋势上

### 4.2 状态管理

- 单一 Zustand store 管理全部游戏状态
- setInterval 驱动 tick 播放
- localStorage 持久化存档

### 4.3 部署

- Vite 构建，`base: '/RetailInvestor/'`
- `npm run deploy` → gh-pages 部署到 GitHub Pages
- 线上地址：https://zjuzwt.github.io/RetailInvestor/

## 5. 开发命令

```bash
npm install       # 安装依赖
npm run dev       # 开发服务器
npm run build     # 生产构建
npm run deploy    # 部署到 GitHub Pages
```
