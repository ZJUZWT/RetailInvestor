# 散户大冒险 - 功能需求文档

> 最后更新: 2026-03-10

## 1. 游戏核心

| 项目 | 描述 |
|------|------|
| 类型 | 炒股主题 Roguelike 网页游戏 |
| 模式 | 单股博弈，以天为回合制 |
| 风格 | 娱乐向，半写实 → 无厘头 → 魔幻三级混合 |
| 目标 | 让人快乐，容易分享 |

## 2. 已实现功能 (v1)

### 2.1 回合系统

- [x] 六阶段日程：晨报 → 上午交易 → 午休 → 下午交易 → 盘后活动 → 结算
- [x] 按天推进，每日结算扣生活费
- [x] 胜利条件：总资产达到目标金额
- [x] 失败条件：总资产不足以支付生活费（破产）

### 2.2 交易系统

- [x] A股规则：涨跌停 ±10%、T+1
- [x] 仓位管理：买入/卖出/观望，支持全仓/半仓
- [x] 实时分时图：241 tick/天，8 时段波动特征
- [x] 播放速度可调：暂停/1x/2x/3x
- [x] 休市后可切换日K/周K/月K/5日线

### 2.3 股价模拟

- [x] 预生成 3-5 个趋势段（牛/熊/震荡），倾向交替
- [x] 日内分时：gravity + noise + trendPush + momentum 算法
- [x] 8 时段特征：开盘博弈 → 主力试盘 → ... → 尾盘疯狂

### 2.4 事件系统

- [x] 三级稀有度：常见(60%) / 稀有(30%) / 传说(10%)
- [x] 32 个事件（18 常见 + 10 稀有 + 4 传说）
- [x] 事件效果：利好/利空/财务/卡牌/体力/事件链
- [x] 事件链机制（CEO 微博系列）

### 2.5 卡牌系统

- [x] 被动能力卡 + 主动能力卡
- [x] 8 张卡牌（4 白 + 3 蓝 + 1 金）
- [x] 卡牌槽位限制（初始 3 个，最多 5 个）

### 2.6 活动系统

- [x] 5 种盘后活动：刷社交 / 复盘研究 / 社交应酬 / 搞副业 / 早点休息
- [x] 体力系统：每日 3 点，午休固定消耗 1 点

### 2.7 目标系统

- [x] 5 个目标：奶茶店(50万) / 丈母娘买房(200万) / 冰岛极光(30万) / 兰博基尼(500万) / 财务自由(300万)

### 2.8 存档

- [x] localStorage 本地存档
- [x] 自动存档 + 继续游戏

### 4.6 精力/饥饿/SAN值系统（重大重构）

**核心理念：** 将游戏从"天为回合制"改造为"时间连续流动 + 角色属性管理"模式。

#### 4.6.1 时间系统改造

- [ ] 取消"进入下一天"的硬切换，时间连续流动
- [ ] 玩家可控制时间流速（暂停/1x/2x/3x），分时图持续播放
- [ ] 日历系统：简单日期+星期显示，标注交易日/休市日
- [ ] 周末自动休市，时间照流（可做盘后活动、恢复属性）
- [ ] 日期从 2026-01-05（周一）开始

#### 4.6.2 三大角色属性

| 属性 | 范围 | 自然衰减 | 恢复方式 | 归零后果 |
|------|------|---------|---------|---------|
| ⚡ 精力 | 0-100 | 清醒时每小时约-4 | 睡觉（非线性：越多越好） | 猝死 💀 |
| 🍚 饱腹 | 0-100 | 每小时约-5 | 吃东西（花钱买食物） | 饿死 💀 |
| 🧠 SAN值 | 0-100 | 缓慢自然恢复+0.5/h | 睡觉+2/h、赚钱微恢复 | 先失控→持续低则精神崩溃 |

#### 4.6.3 睡觉机制

- [ ] 玩家主动选择睡觉，选择睡多少小时
- [ ] 睡觉时暂停玩家操作，时间加速流过（不跳变，提供加速选项）
- [ ] 精力恢复非线性：睡2h恢复少，睡6h恢复多，睡8h近乎满（边际收益递增）
- [ ] 睡觉期间股票照常波动，玩家无法操作
- [ ] 可"提前醒来"但恢复量按实际睡眠时长算

#### 4.6.4 吃饭机制（v1简单版）

- [ ] 固定几种食物：方便面(¥8/+25)、外卖(¥30/+50)、下馆子(¥80/+75)、零食(¥15/+15)
- [ ] 花钱买食物，立即恢复饱腹值
- [ ] 后续可丰富为道具/卡牌系统

#### 4.6.5 SAN值机制

- [ ] 亏损/看跌盘降SAN（看盘时影响加倍）
- [ ] 赚钱/好事件/睡觉回SAN
- [ ] SAN<20 进入失控状态：有概率阻止交易操作（手抖点不了按钮、脑子空白等）
- [ ] SAN=0 + 持续失控 → 精神崩溃 Game Over

#### 4.6.6 死亡条件扩展

- [ ] 猝死：精力归零（连续不睡觉）
- [ ] 饿死：饱腹归零（连续不吃东西）
- [ ] 精神崩溃：SAN归零 + 失控状态持续
- [ ] 破产：保持原有逻辑
