# 散户大冒险 - 功能需求文档

> 最后更新: 2026-03-09

## 1. 游戏核心

| 项目 | 描述 |
|------|------|
| 类型 | 炒股主题 Roguelike 网页游戏 |
| 模式 | 单股博弈，以天为回合制 |
| 风格 | 娱乐向，半写实 → 无厘头 → 魔幻三级混合 |
| 目标 | 让人快乐，容易分享 |

## 2. 已实现功能 (v1)

### 2.1 回合系统

- [x] 六阶段日程：晨报 → 上午交易 → 午休 → 下午交易 → 盘后活动 → 结算
- [x] 按天推进，每日结算扣生活费
- [x] 胜利条件：总资产达到目标金额
- [x] 失败条件：总资产不足以支付生活费（破产）

### 2.2 交易系统

- [x] A股规则：涨跌停 ±10%、T+1
- [x] 仓位管理：买入/卖出/观望，支持全仓/半仓
- [x] 实时分时图：241 tick/天，8 时段波动特征
- [x] 播放速度可调：暂停/1x/2x/3x
- [x] 休市后可切换日K/周K/月K/5日线

### 2.3 股价模拟

- [x] 预生成 3-5 个趋势段（牛/熊/震荡），倾向交替
- [x] 日内分时：gravity + noise + trendPush + momentum 算法
- [x] 8 时段特征：开盘博弈 → 主力试盘 → ... → 尾盘疯狂

### 2.4 事件系统

- [x] 三级稀有度：常见(60%) / 稀有(30%) / 传说(10%)
- [x] 32 个事件（18 常见 + 10 稀有 + 4 传说）
- [x] 事件效果：利好/利空/财务/卡牌/体力/事件链
- [x] 事件链机制（CEO 微博系列）

### 2.5 卡牌系统

- [x] 被动能力卡 + 主动能力卡
- [x] 8 张卡牌（4 白 + 3 蓝 + 1 金）
- [x] 卡牌槽位限制（初始 3 个，最多 5 个）

### 2.6 活动系统

- [x] 5 种盘后活动：刷社交 / 复盘研究 / 社交应酬 / 搞副业 / 早点休息
- [x] 体力系统：每日 3 点，午休固定消耗 1 点

### 2.7 目标系统

- [x] 5 个目标：奶茶店(50万) / 丈母娘买房(200万) / 冰岛极光(30万) / 兰博基尼(500万) / 财务自由(300万)

### 2.8 存档

- [x] localStorage 本地存档
- [x] 自动存档 + 继续游戏

### 2.9 部署

- [x] GitHub Pages 部署
- [x] 访问链接：https://zjuzwt.github.io/RetailInvestor/
