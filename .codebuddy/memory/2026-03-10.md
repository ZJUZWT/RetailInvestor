# 2026-03-10

## 统一时间引擎大重构

### 改动概要
对散户大冒险游戏进行了核心架构重构：

1. **移除 `GamePhase` 概念** — 不再有 `morning_news`、`am_trading`、`lunch_break`、`pm_trading`、`after_hours`、`settlement` 6个阶段切换
2. **统一全局时间引擎** — `globalTick()` 每tick推进1游戏分钟，时间连续流动24小时
3. **股市由 CalendarSystem 的 `marketPhase` 驱动** — 09:30-11:30 上午交易、11:30-13:00 午休、13:00-15:00 下午交易、其他时间盘前/盘后
4. **信息瀑布** — 晨报(08:30)、开盘、午间消息、下午开盘、收盘消息全部自动作为消息推送到信息流
5. **活动常驻** — 活动面板不再局限于"盘后"，任何时间都可用
6. **新增 PnLPanel** — 盈亏面板常驻展示总资产、目标进度、持仓盈亏
7. **结算自动化** — 收盘时自动结算当天损益，新一天自动开始事件
8. **交易面板** — 交易时段显示完整买卖界面，非交易时段显示休市提示+速度控制
9. **速度控制** — 始终显示在交易面板，控制全局时间流速

## 活动系统改用精力(energy)代替体力(stamina)

### 改动概要
活动消耗从体力(stamina)改为精力(vitality.energy)：
- `Activity.staminaCost` → `Activity.energyCost`
- 精力值范围 0-100，活动消耗调整为：刷社交媒体(10)、复盘研究(15)、社交应酬(25)、搞副业(15)
- `doActivity` 直接扣减 `vitality.energy`，不再使用 `stamina` 字段
- ActivityPanel UI 显示精力值，精力不足时红色提示

### 修改文件

## 上班系统（摸鱼/带薪拉屎/离职/找工作）

### 核心机制
- 玩家默认有工作（随机职位+日薪250-500），每个工作日18:00发工资
- 工作时间 09:00-18:00（与股市重叠），上班时要交易必须开启**摸鱼模式**
- 摸鱼每分钟约1.5%概率被领导抓到 → 罚款¥500 + 自动关闭摸鱼
- **带薪拉屎**：每天一次，30分钟内摸鱼免疫，一旦使用不论时长都算消耗掉
- 可以**离职**（自由交易但没工资），可通过活动「找工作」重新就业（50%成功率）
- 初始资金从 ¥30000 降低到 ¥10000（因为有工资收入）

## 上班系统增强：工作进度 + 盘面遮盖

### 核心改动
1. **工作进度系统**：上班时间认真工作（不摸鱼）会累积工作进度(0-100%)，每分钟增加 100/540 ≈ 0.185%
2. **日薪按进度比率发放**：下班时工资 = dailySalary × (workProgress / 100)，摸鱼越多工资越少
3. **不摸鱼时盘面完全不可见**：上班时间 + 有工作 + 不摸鱼 → 左侧面板(K线/交易/盈亏)被遮罩覆盖，只能看到"认真工作中"的提示，可以直接从遮罩上点击开始摸鱼

### 新增字段（JobState）
- `workProgress: number` — 今日工作进度 0-100
- `workMinutesToday: number` — 今日上班总分钟数
- `slackMinutesToday: number` — 今日摸鱼分钟数

### 修改文件
- `src/types/index.ts` — JobState 新增 3 个字段
- `src/stores/gameStore.ts` — globalTick 中工作进度累积逻辑 + 日薪按比率发放 + handleNewDay 重置 + WORK_TOTAL_MINUTES 常量
- `src/components/JobPanel.tsx` — 展示工作进度条、预计薪资、摸鱼分钟统计
